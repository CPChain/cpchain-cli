# CPChain CLI 低代码平台

低代码平台任务：

- 后端事件同步
  - Dependencies 引用其它的合约数据
  - **Stats 统计接口**
- 智能合约生成
  - ERC20 自动发布
  - **基于 AI 实现智能合约无代码设计**
- 生成合约客户端调用代码
  - Step 1: 实现基本的 viewer 和 caller 方法调用，前者不改变状态，后者会改变状态（需要上链）
  - **Step 2: 分析 AST，辨别合约中的实体和动作，分析各个动作对实体的改变以及发出的对应事件；需能够封装好实体及交易的存储**
  - Step 3: 基于 ES6，实现数据的双向绑定（处理好数据的中间状态）
- 前端低代码生成（声明式编码）
